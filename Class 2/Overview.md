### Module Outline
* *Input / Output*: cin, cout, syntax examples, multiple values in cout, common mistakes.

* *Variables & Data Types*: int, float, long long, double, char, bool, variable naming rules, type casting.

* *Operators*: Arithmetic operators (+, -, *, /, %), relational operators (==, !=, <, >, <=, >=), logical operators (&&, ||, !), assignment operators (=, +=, -=, *=, /=), increment/decrement (++, --), operator precedence.
