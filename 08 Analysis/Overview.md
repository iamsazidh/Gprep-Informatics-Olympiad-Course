### Class Outline

* *Complexity Analysis*: Time complexity, space complexity, Big-O notation, constant time `O(1)`, linear time `O(n)`, logarithmic time `O(log n)`, quadratic time `O(n²)`, nested loop analysis.

* *Logarithm Idea*: Basic logarithm intuition, repeated division concept, relation between logarithm and Binary Search, understanding why `log n` appears in algorithms.

* *Constraint Analysis*: Reading input constraints, estimating operation limits, selecting suitable algorithms based on constraints, identifying brute force limits.
