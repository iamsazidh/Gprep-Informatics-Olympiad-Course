# Grep Informatics Olympiad Course

A beginner-friendly roadmap for students preparing for Informatics Olympiad, Competitive Programming, and problem-solving using C++.

---

| Module | Topic | Contents |

| 05 | Arrays | 1D Array, 2D Array, Basic Operations |

| 09 | Brute Force | Basic Brute Force Techniques and Thinking |
| 10 | Vector | STL Vector, Iteration, Dynamic Arrays |

| 31 | Interactive Problems | Interactive Problem Solving Basics |

# 🎯 Course Goals

- Build strong programming fundamentals
- Develop problem-solving intuition
- Learn Competitive Programming techniques step-by-step
- Prepare for Informatics Olympiad and ICPC-style contests

---

# 🛠 Language Used

- C++
- GNU G++ Compiler
- STL (Standard Template Library)

---

# 🌐 Recommended Online Judges

- Codeforces
- AtCoder
- CSES
- LightOJ
- UVA
- SPOJ

---

# 🚀 Target Audience

- School & College Students
- Beginner Competitive Programmers
- Informatics Olympiad Aspirants

---

# 📌 Notes

- Every module contains explanations, examples, and practice problems.
- The course is designed progressively from beginner to intermediate level.
- Focus on understanding concepts instead of memorizing solutions.

---

# ⭐ Advice

Consistency beats motivation in Competitive Programming.

Solve problems daily, even if it is just one problem.
Small progress compounds insanely over time.
